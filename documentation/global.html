add(num1, num2) → {number}

This function accepts two numbers as parameters and returns their sum. If the passed parameters are not numbers, NaN is returned
Parameters:
Name Type Description
num1 number The first number to be added
num2 number The second number to be added
Returns:
num1 + num2
Type
number
Example
add(1,3)
//returns 4
add(9,0)
//returns 9
add(-4, 23)
//returns 19

Name Type Description

num2 number The second number that is the Divisor

Airthmatic Exception when second parameter is 0

num1/num2

number
Example

//returns 4

//returns 0

validateNumber(num) → {boolean}

This helper function accepts one parameter and checks if that object can be a valid number or not. The method toString() is called automatically before the check
Parameters:
Name Type Description
num Object | string Any object to be checked for number validation
Returns:
Type
boolean
Example
validateNumber("1292")
//returns true
validateNumber("928.8292")
//returns true
validateNumber("number")
//returns false
validateNumber({k1: v1})
//returns false
